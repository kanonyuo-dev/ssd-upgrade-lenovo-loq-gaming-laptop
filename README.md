## Overview (TL;DR)

A Lenovo LOQ Full HD Gaming Laptop storage drive was upgraded from a 512 GB NVMe SSD to a 1 TB WD_BLACK SN7100 NVMe SSD to increase storage capacity and maintain high-speed performance. The OS was migrated successfully, and the system passed post-upgrade validation.

---

## Background / Context

The user reported:

- Storage nearing capacity  
- Need for additional space for games and applications  
- Desire to maintain high NVMe performance  

---

## Problem Statement

The existing 512 GB SSD provided insufficient storage for the user’s growing workload and application needs.

---

## Environment & Tools

### Device Information

- **Device:** Lenovo LOQ Full HD Gaming Laptop  
- **OS:** Windows 11  

### Hardware Used

- SanDisk WD_BLACK SN7100 1TB Internal SSD NVMe

### Tools Used

- Phillips #00 Screwdriver  
- Phillips #000 Screwdriver  
- Flashlight  
- Tweezers  
- Triangle Prying Picks  
- SSK M.2 NVMe SATA SSD Enclosure Adapter  

---

## Troubleshooting Process

1. Verified available disk space and user storage requirements.  
2. Confirmed NVMe compatibility with the laptop motherboard.  
3. Connected new SSD via NVMe enclosure adapter.  
4. Migrated Windows 11 installation to the new drive.  
5. Powered down system and opened chassis.  
6. Removed original 512 GB NVMe SSD.  
7. Installed WD_BLACK SN7100 1 TB SSD.  
8. Reassembled device and booted system.  
9. Verified successful boot and full drive recognition.  
10. Checked disk health and performance in Windows.

---

## Root Cause

The original 512 GB SSD lacked sufficient capacity for the user’s storage demands.

---

## Resolution

- Upgraded storage from **512 GB → 1 TB NVMe SSD**  
- Successfully migrated Windows 11  
- Validated drive health and performance  

---

## Results / Impact

- Doubled available storage capacity  
- Maintained high NVMe performance  
- Eliminated low-storage warnings  
- Improved user workflow and storage flexibility  

---

## Lessons Learned

- Always verify NVMe compatibility before upgrade.  
- External enclosures simplify OS migration workflows.  
- Post-migration boot validation is critical to success.
